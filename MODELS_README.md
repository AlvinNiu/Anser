# 免费3D模型资源指南

## 🆓 免费USDZ模型下载网站

### 1. Apple USDZ Gallery（推荐）
**网址**: https://developer.apple.com/augmented-reality/quick-look/
- 苹果官方提供的免费USDZ模型
- 质量高，优化好
- 点击模型可直接下载

### 2. Sketchfab（免费筛选）
**网址**: https://sketchfab.com/search?type=models&downloadable=true
- 筛选 "Downloadable" 标签
- 格式选择 USDZ 或 OBJ（可转换）
- 大量免费食物模型

### 3. Poly Pizza（原Google Poly）
**网址**: https://poly.pizza
- 低多边形风格
- 免费下载
- 适合休闲游戏风格

### 4. CGTrader（免费专区）
**网址**: https://www.cgtrader.com/free-3d-models
- 搜索 "food" + 免费筛选
- 格式丰富

### 5. Free3D
**网址**: https://free3d.com
- 免费食物模型专区
- 需要注册

## 🛠️ 模型下载步骤

### 以 Sketchfab 为例：
1. 访问 https://sketchfab.com/search?q=apple&type=models
2. 左侧筛选栏勾选 "Free" 和 "Downloadable"
3. 点击喜欢的模型
4. 点击 "Download 3D Model"
5. 格式选择 USDZ (如果没有，选 OBJ 用 Reality Converter 转换)
